# 🚀 gemini-chat-backend

API backend do projeto de chat com integração futura com Gemini.

---

## 🧱 Stack

- Node.js
- NestJS
- Prisma ORM
- PostgreSQL (Neon)
- TypeScript

---

## ▶️ Como rodar

```bash
# instalar dependências
npm install

# gerar client do prisma
npx prisma generate

# rodar migrations
npx prisma migrate dev

# iniciar aplicação
npm run start:dev
